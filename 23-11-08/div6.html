텍스트 태그와 css

          < b > 글씨 굵게 < /b >
          < strong > 글씨 굴게(문장에서 강조용) < /strong >
          < h1~6 > 글의 제목 < /h1 >
          < p > 문단의 표현 < /p >
          < br > 텍스트 줄바꿈
          < blockquote > 텍스트의 인용문 < /blockquote >
          < q > 텍스트의 인용문(줄바꿈 없음) < /q >
          < pre > /pre > 
텍스트에 서식지정(작성한 내용 그대로 표현)
 < /pre >
          < figure > < /figure > 
          < hr > 장면전환, 구획내 주제변경, 문단 분리
 
          < i > < /i > 이탤릭체, 기울임
          < em > < /em > 텍스트의 강조, 중첩사용가능
          < mark > < /mark > 텍스트 중요표시, 하이라이트 표시
          < small > < /small > 덧붙이는 글, 저작권과 법률 같은 것
          < sup > < /sup > 위 첨자
          < sub > < /sub > 아래 첨자
          < del > < /del > 제거 텍스트
          < ins > < /ins > 추가된 텍스트
          < u > < /u > 텍스트에 밑줄
          < kdb > < /kdb > Ctrl+R
        
          font-size:20px; 글씨 크기를 20px로 설정
          font-weight:700; 글씨 굵기 설정 100단위
          font-family:'궁서'; 글꼴 설정
          color:red; 글씨 색상
          text-decoration:underline; 글씨장식 설정
          text-transform:lowercase; 대소문자 변환
          text-align:center; 텍스트 정렬(수평정렬)
          letter-spacing:-2px;글자간격
          line-height:30px; 행 높이
